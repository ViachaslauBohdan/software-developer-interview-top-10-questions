# Python vs JavaScript — code examples

Each folder has **`main.py`** and **`main.js`** demonstrating the same idea in both languages.

| Folder | Topic |
|--------|--------|
| `01-variables-and-types/` | Names, assignment, `None` vs `null`/`undefined` |
| `02-collections/` | list/dict/set vs Array/object/Map/Set |
| `03-functions/` | defaults, `*args`/`**kwargs` vs rest/spread |
| `04-classes/` | classes, `self` vs `this` |
| `05-modules/` | import patterns (conceptual) |
| `06-iteration/` | `for`, comprehensions vs `map`/`filter` |
| `07-errors/` | `try`/`except` vs `try`/`catch` |
| `08-async/` | `asyncio`/`await` vs `async`/`await` + Promises |
| `09-equality-truthiness/` | `==`, truthy/falsy |

Run from repo root:

```bash
python3 python/code_examples/01-variables-and-types/main.py
node python/code_examples/01-variables-and-types/main.js
```
