# JavaScript `Math` Reference

Purpose-focused examples for popular `Math` constants and functions.

## Files

1. `01-math-functions-with-outputs.js`
- Constants (`PI`, `E`, etc.)
- Rounding (`floor`, `ceil`, `round`, `trunc`)
- Basic operations (`abs`, `min`, `max`, `pow`, `sqrt`, `cbrt`)
- Trigonometry (`sin`, `cos`, `tan`, `asin`, `acos`, `atan`, `atan2`)
- Logs / exponentials (`log`, `log10`, `log2`, `exp`, `expm1`, `log1p`)
- Utility / numeric helpers (`random`, `sign`, `hypot`, `imul`, `clz32`, `fround`)

Run with:

```bash
node 01-math-functions-with-outputs.js
```
